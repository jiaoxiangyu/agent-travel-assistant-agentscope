- **推荐时长**：4-6天（原3天2晚调整）
- **旅行风格**：历史人文+博物馆+地标+胡同体验，侧重文化互动与浪漫场景
- **预算分配**：总额¥3000（不含住宿）  
  - 门票：通过旅游年卡（¥190）覆盖60+景点  
  - 交通：建议购买80次地铁卡（¥160）  
  - 餐饮：¥1200（占40%），含胡同美食/宫廷菜系/手作体验
- **成本等级**：中等偏高，可选年卡节省开支

## 行程规划
### 文化景点
- **必访场所**：故宫/天坛/颐和园/国家博物馆（需提前3天通过「北京本地宝」预约）
- **特色体验**：
  - 胡同漫步+冰窖餐厅晚餐
  - 南锣鼓巷手作体验/前门大街创意料理
  - 故宫红墙夜游/景山观城项目
  - 什刹海钟鼓楼夜景+护国寺小吃
- **替代方案**：突发降雨时转为博物馆/798艺术区等室内活动

### 交通指南
- **主要方式**：地铁网络（覆盖核心区域）
- **辅助工具**：短途打车应对高峰时段
- **重点站点**：天安门东站/北宫门站/王府井站
- **避峰建议**：避开早高峰时段前往热门景区
- **预约提示**：部分场馆安检+预约耗时较长，预留充足时间
