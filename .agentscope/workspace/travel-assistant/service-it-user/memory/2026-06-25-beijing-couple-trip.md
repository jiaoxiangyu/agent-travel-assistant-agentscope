# 北京3天2晚情侣游行程草案

## 城市画像
- **推荐时长**：4–6天（本次为紧凑型3天）
- **风格**：历史人文、博物馆、城市地标、胡同体验
- **交通**：地铁覆盖广，热门景区安检和预约耗时较长
- **消费水平**：中等偏高
- **提示**：故宫、国家博物馆等建议提前1–7天实名预约；行程不宜过满；夏季注意防晒防暑。

## 实时天气
- **城市**：北京
- **天气**：晴天
- **气温**：32.3°C
- **湿度**：24%
- **风速**：6.7米/秒
- **提示**：高温干燥，注意防晒补水，避免正午长时间户外活动。

## 景点候选（人文类）
1. 故宫
2. 天坛
3. 颐和园
4. 国家博物馆

## 结构化行程草案

### 第1天：故宫 + 天坛 + 什刹海夜景
- **上午**：参观故宫（建议从午门进，神武门出），重点游览中轴线三大殿与东西六宫。
- **下午**：前往天坛，游览祈年殿、回音壁、圜丘坛。
- **晚上**：返回市区，漫步什刹海或南锣鼓巷，感受老北京夜生活。

### 第2天：颐和园深度游 + 胡同休闲
- **上午**：游览颐和园，建议从北宫门进，经苏州街、仁寿殿至佛香阁。
- **下午**：沿长廊步行至昆明湖，可乘船或环湖散步。
- **晚上**：返回前门或鼓楼附近，品尝老北京小吃或私房菜。

### 第3天：国家博物馆 + 文艺收尾
- **上午**：参观国家博物馆（需提前预约），重点看古代中国展。
- **下午**：视体力选择补游或前往五道营胡同，逛文创店、咖啡馆。
- **傍晚**：灵活安排返程。

## 注意事项
- 所有门票、营业时间、预约要求请以官方实时信息为准。
- 夏季高温，建议清晨出行，午后可安排室内场馆或咖啡厅休息。
- 行程可根据体力和兴趣灵活调整，避免过度疲劳。
