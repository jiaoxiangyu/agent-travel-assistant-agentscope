## 旅行目的地：北京情侣游（3天2晚）

### 概览
- **旅行主题**：历史人文与浪漫体验结合的情侣短途旅行。
- **时间安排**：3天2晚，节奏适中，兼顾景点游览与休闲放松；若时间充裕可延长至4–6天以更深度体验。
- **人均预算**：约3000元，涵盖餐饮、市内交通、门票及纪念品；**住宿费用另计**（建议两晚总预算800–1200元）。
- **推荐区域**：东城区或西城区（如王府井、前门、鼓楼、五道营胡同周边），便于游览核心景点且交通便利，兼具文化氛围与生活便利性。

### 住宿建议
- **区域选择**：优先考虑东城或西城区的胡同文化圈，如鼓楼、五道营胡同、前门或王府井附近，步行可达多个景点，夜生活丰富。
- **预算范围**：两晚控制在800–1200元，推荐精品民宿、连锁酒店或四星级标准住宿，兼顾舒适性与地理位置。
- **优选类型**：具有老北京四合院风格的精品客栈或设计型文艺民宿，提升沉浸感和拍照体验。

### 行程规划

#### 第1天：故宫 + 冰窖餐厅 + 天坛 + 什刹海夜景
- **上午**：参观故宫（需提前预约），建议从午门进入，神武门出；重点游览太和殿、乾清宫、东西六宫及珍宝馆。**建议清晨出发**，避开人流高峰与高温。
  - **升级体验**：可在冰窖餐厅享用轻奢午间简餐，融合宫廷元素与现代风味。
- **下午**：前往天坛公园（提前预约购票），游览祈年殿、回音壁、圜丘坛，感受古代祭祀文化。
- **晚上**：漫步什刹海，推荐银锭桥观赏落日与城市夜景；可在烟袋斜街或后海北沿清吧小酌，享受情侣静谧时光。南锣鼓巷可视兴趣短停留，但主街较拥挤，建议走支巷探索。

#### 第2天：颐和园深度游 + 胡同休闲
- **全天**：游览颐和园（建议提前预约），推荐路线：北宫门进 → 苏州街 → 仁寿殿 → 长廊 → 佛香阁 → 排云殿 → 昆明湖。
- **特色体验**：
  - 登佛香阁俯瞰全园，或乘船泛舟昆明湖。
  - **升级选项**：可选择仿古龙舟体验，增强皇家园林的仪式感与浪漫氛围。
- **傍晚**：返回鼓楼或五道营胡同区域，品尝本地风味晚餐，如铜锅涮肉（聚宝源、南门涮肉）、私房菜或融合菜。

#### 第3天：国家博物馆 / 景山公园 + 文艺收尾
- **上午选项一（文化控推荐）**：参观国家博物馆（必须提前预约），重点参观“古代中国”基本陈列展，系统了解中华文明脉络。
- **上午选项二（摄影党推荐）**：前往景山公园登顶万春亭，俯瞰紫禁城全景，是拍摄故宫中轴线的绝佳机位。
- **下午**：前往五道营胡同，文艺气息浓厚，适合慢逛文创店、独立咖啡馆（如METAL HANDS）、小众书店与手作工坊，轻松结束旅程。
  - **推荐消费**：单人饮品30–50元，可搭配甜点或轻食，整体性价比高。
- **傍晚**：根据返程交通预留缓冲时间，建议提前查询地铁/高铁/机场快线时刻表。

> ⚠️ **动线说明**：第1天整合故宫、天坛形成完整的历史轴线；若体力有限，可将天坛调整至第3日上午替代国博，实现“南中轴+老城风貌”主题闭环。
